Estudando HTML

Nossos estudos começam falando de um item muito sério que é a organização. Isso porque a linguagem responsavel apenas pela marcação dos elementos na janela do navegador. o HTML não cria nada, ele manda buscar letrinhas que voce digita

Parágrafos

Para colocarmos parágrafos, usamos os elementos "p" e "/p". Para fazer uma quebra de linha, mantendo o mesmo paragrafo, usamos o elemtno "br". Temos ainda parágrafos especiais denominados titulos, cabeçalhos ou headings em que os elementos de "h1" a "h6" e seus respectivos fechamentos "/h1" a "/h6"

Imagens

Uma imagem aparece na janela do navegador pelo uso do elemento "img". Esse elemento possui o atributo "src", responsavel pela localização da imagem. Para trabalhar melhor com a imagem, voce pode colocá-la dentro das tags de marcação de parágrafo.
